# NovaOS Apps Layer

The **Apps** layer contains core and system applications designed for NovaOS.

### Goals
- Build lightweight, privacy-friendly apps.
- Offer open-source replacements for Google apps.
- Allow APK sandboxing for Android compatibility.

### Example apps
- NovaStore (app store for open-source apps)
- NovaBrowser (privacy-first browser)
- NovaDialer, NovaMessages, NovaCamera
