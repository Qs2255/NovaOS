# Contributing to NovaOS

Thank you for considering contributing to **NovaOS**!

## How to contribute
1. Fork the repo and create a new branch.
2. Follow coding standards in `docs/README.md`.
3. Submit a pull request explaining your change.
4. Be respectful and responsive during review.

## Communication
Join discussions on GitHub or the future NovaOS community page.

## Reporting Bugs
Use the “Issues” section with the prefix `[BUG]` and steps to reproduce.
