NovaOS

Open-source mobile OS — privacy first, user owned.

What is NovaOS

NovaOS is a community-driven operating system for mobile devices built from a Linux foundation with a secure sandbox for running Android apps when needed. It is designed for transparency, security, and complete user control — no ads, no trackers, no hidden telemetry.

Highlights

  • Privacy by design — telemetry disabled by default
  • Safe, auditable root access under user control
  • Android APK compatibility via isolated runtime
  • Reproducible builds and community governance
